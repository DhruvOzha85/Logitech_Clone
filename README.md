# 🖱️ Logitech Clone

A static front-end clone of the **Logitech** official website, built with pure HTML and CSS. This project recreates key sections of Logitech's site including their products, software, support, sustainability (Planet), and about pages.

🔗 **Live Demo:** [clonelogitech.netlify.app](https://clonelogitech.netlify.app/)

---

## 📁 Project Structure

```
Logitech_Clone/
├── index.html       # Home page
├── products.html    # Products listing page
├── software.html    # Software page
├── support.html     # Support / Help page
├── planet.html      # Sustainability / Planet page
├── about.html       # About Logitech page
└── styles.css       # Global stylesheet
```

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure & content (80.8%) |
| CSS3 | Styling & layout (19.2%) |

---

## ✨ Features

- 6-page multi-section static website
- Covers all major navigation sections of the Logitech website
- Clean product-focused layout and styling
- Includes a dedicated sustainability/planet page
- Deployed and publicly accessible via Netlify

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/DhruvOzha85/Logitech_Clone.git
   ```
2. Open `index.html` in your browser — no build step or dependencies required.
